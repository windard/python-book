os

os

非常基础的一个库，但是却实现了我一个想了很久了功能，识别目录下的所有文件。

  1. 取得当前目录-- os.getcwd
  2. 更改当前目录-- os.chdir
  3. 创建一个目录-- os.mkdir
  4. 创建多级目录-- os.makedirs
  5. 删除一个目录,只能删除空目录-- os.rmdir
  6. 删除多个目录,删除目录及其下内容-- os.removedirs
  7. 获取目录中的文件及子目录的列表-- os.listdir

    隐藏文件也会显示出来

  8. 删除一个文件-- os.remove
  9. 删除一个文件-- os.unlink
  10. 文件或者文件夹重命名-- os.rename(old， new)
  11. 获取文件大小-- os.path.getsize
  12. 获取文件属性-- os.stat
  13. 修改文件权限与时间戳-- os.chmod
  14. 路径中加入新的内容-- os.path.join(path,file)
  15. 将路径分解为目录名和文件名-- os.path.split
  16. 将目录分解为目录加文件名和文件名的扩展名-- os.path.splitext
  17. 获得路径的路径名-- os.path.dirname
  18. 获得路径的文件名-- os.path.basename
  19. 判断一个路径是否存在或是否为路径-- os.path.isdir
  20. 判断一个文件是否存在或这否为文件-- os.path.isfile
  21. 判断一个路径（目录或文件）是否存在——os.path.exists
  22. 判断一个路径是否是绝对路径-- os.path.isabs
  23. 获得一个绝对路径 -- os.path.abspath 或者 os.path.realpath

    获得当前文件所在的文件夹 os.path.abspath(os.path.dirname(__file__)) 或者 os.path.dirname(os.path.abspath(__file__)) 或者 os.path.abspath(os.curdir)

  24. 获得一个相对路径 -- os.path.relpath(path, start)
  25. 读取和设置环境变量-- os.getenv 与 os.putenv
  26. 获取家目录-- os.path.expanduser('~')
  27. 指示你正在使用的平台-- os.name

    对于Windows，它是nt，而对于Linux/Unix/Mac用户，它是posix

  28. 给出当前平台使用的路径分隔符 -- os.sep

    Windows 使用 \ Linux 使用 /，Mac OS 使用 /

  29. 给出当前平台使用的行终止符-- os.linesep

    Windows 使用 \r\n,Linux 使用 \n,低版本 Mac OS 使用 \r，高版本 Mac 用 \n

  30. 给出当前平台使用的分割路径的分隔符 -- os.pathsep

    Windows 使用 ; ，Linux 使用 : Mac OS 使用 :

  31. 运行shell命令-- os.system

    但是这个执行命令行没有返回值，直接输出，不管你有没有print

  32. 执行shell命令-- os.popen

    执行命令行，返回一个file open的对象，需要read才能得到执行结果，但是还是没有返回值，如果需要更多的命令行操作，可以使用commands库

  33. 执行 shell 命令 -- os.execlp

    如 os.execlp('ls','') 并替换当前进程

  34. 终止当前进程-- os._exit

    比如 os._exit(0) 还能从子线程中终止主线程。sys.exit 则不能这样

  35. 循环遍历目录-- os.walk

    返回一个三元组，第一个是路径，第二个是路径下的目录，第三个是路径下的非目录。

  36. 系统环境变量-- os.environ

    返回系统环境变量，或者是在 HTTP 请求中的请求头。

  37. 生成随机字符串-- os.urandom

    比如 os.urandom(num) 返回 num 个随机字符串，在 ASCII 中，不一定是可打印字符。

  38. 检查文件系统权限-- os.access(filename, privilege)
  39. 创建一个新的进程 -- os.fork

    只在 linux 和 unix 上使用，windows 用不了

  40. 获得当前进程的 pid -- os.getpid
  41. 获得父进程的 pid -- os.getppid
  42. 杀死一个进程 -- os.kill

    比如 os.kill(os.getpid(), 9) 杀死当前进程

# coding=utf-8
import os

def showall(path,leavel=0,filenum=0,show=True):
    newnum = filenum
    currentpath = path;
    dirandfile = os.listdir(path)
    for item in dirandfile:
        newpath = os.path.join(currentpath,item)
        if os.path.isdir(newpath):
            num = showall(newpath,leavel+1,newnum,show)
            newnum = num
        else:
            newnum = newnum + 1
            tab_stop = ""
            if show:
                for tab in range(leavel):
                    tab_stop = tab_stop + " "
            print tab_stop + newpath

    return newnum

if __name__ == '__main__':
    num = showall('./',show=False)
    print "File Number : " + str(num)
